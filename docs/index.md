# Deploy process

Documentation: https://docs.expo.dev/build/introduction/

## Commands cheat sheet:  

### Expo Application Services 
 
`npm install -g eas` 
 
*Don't forget to bump the build version*  
 
`eas build -p [ios/android/all]`  
`eas submit -p [ios/android/all] --latest`
 
### Github Pages
 
`yarn deploy`
